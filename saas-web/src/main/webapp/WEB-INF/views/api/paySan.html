统一下单接口(PC扫码)

应用级输入参数

名称 类型 是否必须 示例值 描述
appKey String(18) 是 100120160907100010 维金分配给商户的唯一身份key
merchantId int(20) 是 100120 商户ID(维金自动生成)
amount String(20) 是 0.01 支付金额，精确到小数点后两位
channelCode String(10) 是 WXPAY 渠道代码，商户所调用的渠道
createTime String(20) 是 2016-07-04 10:05:20 创建支付单时间 yyyy-MM-dd HH:mm:ss
proSubject String(20) 是 商品主题 商品主题，商户自行描述
proDesc String(20) 否 商品描述 商品描述，商户自行描述
instOrderNo String(20) 是 1123434423367991118 外部订单号
toRSA String(100) 是 WASjvAbsOzvgntONsODBWEQm6JHq RSA签名，身份验证

返回值

名称 类型 是否必须 示例值 描述
appKey String(20) 是 维金分配给商户的唯一身份key
merchantId int(20) 是 100120 商户ID(维金自动生成)
channelCode String(10) 是 WXAPP 渠道代码，商户所调用的渠道
errorMsg String(100) 是 错误信息
errorCode String(10) 是 0002 错误代码 目前默认0002无需理会
amount String(5) 是 0.01 支付金额，精确小数点两位小数
mchId String(20) 是 商户在渠道所注册的商户号
url String(200) 是 weixin:/pay/bizpayurl?pr=ibGMgg1 返回二维码URL 微信需要自己生产二维码，支付宝求情该URL即可
orderNo String(20) 是 机构订单号
responseCode String(10) 是 0003 接口响应状态 0003 处理中 0002 失败
responseMsg String(10) 是 PROCESSED 接口响应信息 PROCESSED 处理中 FAIL 失败
